HowTo - use tkbibtex under Windows 98

This may apply to other Windows variants as well

  • install Tcl/Tk on your system
  • install tkbibtex on your system with a filename of tkbibtex.tcl
  • fire up a File browser
  • select View|Folder Options
  • select File Types tab
  • select New Type
  • Description of type := BibTex database
  • Associated extension := BIB
  • select New ...
  • Action := open
  • Application used to perform action := "c:\Program Files\Tcl\bin\wish83.exe" "c:\bin\tkbibtex.tcl" "%1" (note the quotes are required, adjust the paths to suit your installation)

Now you should just be able to double-click on a bibfile and tkbibtex will be invoked on it. There is no icon yet, anybody feeling artistic?
